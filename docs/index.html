Quick start

Up and running in under 2 minutes.

1

Install the extension

Search for "inkpilot" in VS Code Extensions, or run:

code --install-extension Redmond-Enterprises-LLC.inkpilot
2

Connect your MCP client

Claude Code and GitHub Copilot auto-register. Others connect via SSE:

http://localhost:7100/sse
3

Start designing

Open the command palette, run inkpilot: Open SVG Editor, then ask your AI to design something.

> inkpilot: Open SVG Editor
